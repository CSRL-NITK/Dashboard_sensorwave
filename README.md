# SensorWave: IoT Dashboard

SensorWave is an end-to-end IoT monitoring system designed to collect, store, and visualize real-time sensor data while providing secure role-based access to users. It integrates ThingsBoard for device management, PostgreSQL for data storage, Grafana for visualization, and a custom React + Node.js web application.

## Features
- Real-time sensor data collection  
- OAuth 2.0 authentication with role-based access (Admin, Vendor, User)  
- Device and user management  
- Interactive dashboards using Grafana  
- Map-based device visibility with Leaflet  
- Modular and scalable backend architecture  


## Technology Stack
- Frontend: React.js  
- Backend: Node.js (Express)  
- IoT Platform: ThingsBoard  
- Database: PostgreSQL  
- Visualization: Grafana  
- Authentication: OAuth 2.0
